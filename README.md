# 年终大抽奖 2025 - Web版

一个基于 HTML/CSS/JavaScript 的年终抽奖网页应用，具有喜庆的春节风格视觉设计，支持导入名单、烟花庆祝效果和背景音乐播放。

## 功能特色

- 🏮 **红灯笼装饰** - 页面顶部左右两侧，带摆动动画
- 🔴 **红色主题** - 中国红 + 金色配色方案
- 🎆 **烟花特效** - Canvas 粒子系统实现
- 🎵 **背景音乐** - 可控制播放/停止
- 📋 **名单导入** - 支持粘贴或文件上传（TXT/CSV）
- 💾 **结果导出** - 一键导出中奖名单为 TXT 文件
- 🎊 **纸屑效果** - 中奖时的庆祝动画
- ⚡ **滚动动画** - 名字快速滚动后减速停止
- ⭐ **星星装饰** - 背景闪烁星星效果

## 使用方法

### 1. 准备音乐文件（可选）

将背景音乐文件命名为 `bg-music.mp3`，放在与 HTML 同一目录下。
建议使用欢快的节日音乐。

### 2. 运行程序

**方式一：直接打开**
```
双击 index.html 文件，用浏览器打开
```

**方式二：使用本地服务器**
```bash
# Python
python -m http.server 8000

# Node.js (需要先安装 http-server)
npx http-server
```

然后在浏览器访问 `http://localhost:8000`

### 3. 导入名单

1. 点击 **"导入名单"** 按钮
2. 选择以下方式之一：
   - **手动输入**：在文本框中输入名字，每行一个
   - **上传文件**：点击"上传文件"选择 TXT 或 CSV 文件
   - **使用示例数据**：点击"使用示例数据"快速体验

### 4. 开始抽奖

1. 点击 **"开始抽奖"** 按钮
2. 名字会快速滚动并逐渐减速
3. 最终显示中奖者，同时触发烟花和纸屑效果
4. 中奖者会从名单中移除，不会重复中奖

### 5. 导出结果

点击 **"导出结果"** 按钮，会自动下载 TXT 文件，同时复制到剪贴板。

## 名单文件格式

TXT 或 CSV 文件，每行一个名字：

```
张三
李四
王五
赵六
...
```
